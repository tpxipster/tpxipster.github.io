**一、参保条件**

|     |     |
| --- | --- |
| 1   | 本市户籍非在园在校少儿。 |

**二、参保缴费形式及缴费标准**
** （一）参保缴费形式**
**在园在校学生：**按学年一次性缴纳医疗保险费，参保学生应将医疗保险费缴交到学校，学校统一归集。
**非在园在校少儿（深户籍）：**由社保机构一次性从监护人或少儿本人的缴费账户中扣取全年费用。

|     |     |
| --- | --- |
| 1   | 在**2016年4月14日之后**激活了参保人本人的金融社保卡，将自动使用本人的金融社保卡作为缴费账户，未激活使用的将继续使用监护人（大学生为本人）的银行账户缴费； |
| 2   | 在**2016年4月14日之前**激活了本人金融社保卡的，但事后通过银行窗口、社保局个人网页、微信等渠道确认将金融社保卡作为社保账户的，也将自动使用本人的金融社保卡作为缴费账户。 |

符合参保条件，申请参保缴费的学生在接到学校通知后，将本年度医疗保险费统一缴交到所在学校。学校归集所有学生医疗保险费后，由社保经办机构在学校指定账号划扣。

**问题1**
非在园在校少儿的金融社保卡还没办下来，今年9月的少儿医保，是不是还是从监护人的卡上扣取保费？
**答：**是的，金融社保卡还没下来，就从监护人账户上扣款。
**问题2**
在园深户，之前是直接从家长账户扣费，现已办好幼儿的金融社保卡并且已激活，这次的缴费是直接从幼儿账户扣费，还是交给幼儿园统一办理？
**答：**在园在校的宝宝，还是要缴交给幼儿园统一办理。
**问题3**![640.png](https://raw.githubusercontent.com/tpxipster/tpxGalaxy/master/vnote笔记汇/少儿医保统一申报马上开始.md/640-4.png)
深户宝宝上学期在幼儿园参保缴费了，但是这学期不上幼儿园了，可以自己买社保吗？

**答：**深户宝宝可以哦，可通过深圳市人力资源和社会保障局官网（http://hrss.sz.gov.cn/）的个人社保网上服务系统、“深圳人社”微信公众号及社保自助服务终端机，进行参保登记缴费。

**问题4**![640.png](https://raw.githubusercontent.com/tpxipster/tpxGalaxy/master/vnote笔记汇/少儿医保统一申报马上开始.md/640-4.png)
新学期小朋友转到本市其它幼儿园了，少儿医保怎么交？
**答：**由新幼儿园统一收取参保缴费。
**问题5**![640.png](https://raw.githubusercontent.com/tpxipster/tpxGalaxy/master/vnote笔记汇/少儿医保统一申报马上开始.md/640-4.png)
宝宝是深户，7月刚参加少儿医保，扣了一笔费用，当时绑定的是监护人的卡扣款的，九月上幼儿园要怎么操作？
**答：**7月缴纳的费用是2018-2019年度少儿医保费用，9月缴交的是2019-2020年度的费用，孩子上幼儿园的话，就由幼儿园统一收取。
**问题6**
非在园在校的深户孩子，医保费还在大人的银行卡上划扣吗？
**答：**非在园在校少儿及新生儿如果办理并激活了金融社保卡，将自动使用少儿本人的金融社保卡作为缴费账户。
如果金融社保卡仍未激活，将继续使用监护人的银行账户缴费。家长要确认孩子的金融社保卡是否激活，并且在账户里存够钱。
**问题7**![640.png](https://raw.githubusercontent.com/tpxipster/tpxGalaxy/master/vnote笔记汇/少儿医保统一申报马上开始.md/640-4.png)
少儿医保缴费标准去年是309.36元，今年怎么变成了371.64元？

**答：**少儿医保缴费标准与社会平均工资挂钩，缴费标准为本市上年度在岗职工月平均工资×0.8%×12个月，今年最新的社平工资是9309元，少儿医保缴费计算公式为9309*0.8%=74.47元/月，74.47*12个月=893.64元，扣除政府补贴522元/年，参保人实际需缴交医疗保险费371.64元。
